E-olymp User Management

EolympUserManagement is a web application that finds differences between two users using data from the Eolymp platform. This application produces interesting results by processing users' information on Eolymp.

The project is specifically related to the Eolymp platform and analyzes users' data on this platform. Eolymp's usage rights have been respected, and the purpose of the project is for learning and entertainment purposes only.
